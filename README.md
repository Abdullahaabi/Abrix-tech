## üîß Customization

### Colors
To change the color scheme, update the CSS variables in `styles.css`:
```css
:root {
    --primary-bg: #1a1a1a;
    --accent-color: #ff6b00;
    --text-color: #ffffff;
    --secondary-text: #cccccc;
}
```

### Content
- Update text content in `index.html`
- Replace images with your own
- Modify contact information in the footer
- Update testimonials in `script.js`

### Animations
- Adjust animation timing in CSS transitions
- Modify JavaScript animation parameters
- Add or remove hover effects
